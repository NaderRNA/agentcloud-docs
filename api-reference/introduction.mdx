## Welcome

The Agent Cloud API (under construction) enables users to programatically interact with the app. 
This is intended to work on every component of the app enabling CRUD over: 
* Apps
* Agents
* Tools
* Tasks
* Models
* Credentials
* Users
* Workspace

## Authentication

To get started, you need to have a developer token. To generate a token
<Steps>
  <Step title="Login">
    Login to the platform at https://app.agentcloud.dev/login 
  </Step>
  <Step title="Click on Developer">
    Click on the Developer tab (if this is available for your permission)
  </Step>
  <Step title="Generate">
    Click generate which will reveal a secret key which you can copy and use in your code.
  </Step>
</Steps>
